# Medical Email Triage System (NLP)

## 📌 Project Overview
This project automates the classification and prioritisation of patient emails using Natural Language Processing (NLP).

## 🚀 Features
- Extracts structured data from unstructured emails
- Predicts medical conditions using ML models
- Recommends appropriate departments/doctors
- Real-time processing pipeline

## 🛠 Tech Stack
- Python
- Scikit-learn
- NLP (spaCy / transformers)
- Streamlit / Flask

## 📊 Results
- Achieved 85%+ accuracy
- Reduced manual triage time by 80–90%

## 📁 Project Structure
- data/
- notebooks/
- model/
- app/

## ▶️ How to Run
1. Clone the repo
2. Install requirements
3. Run main.py
